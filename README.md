# OmniAI Core

> One interface. Every AI.

`omniai-core` defines the core interfaces, types, and contracts
used across the OmniAIHQ ecosystem.

This package is provider-agnostic and contains **no direct
implementations**.

## Features
- Unified provider interfaces
- Explicit capability detection
- Extensible design for future AI models

## Installation
```bash
npm install @omniaihq/core
