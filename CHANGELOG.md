# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2025-12-29

### Added

- Initial release of `@omniaihq/core`.
- Defined base provider interfaces (`BaseProvider`).
- Defined capability flags.
- Added text and image generation interfaces (`TextProvider`, `ImageProvider`).
- Implemented `ProviderFactory` for dynamic provider registration.
- Added custom error types (`OmniAIError`, `ProviderNotSupportedError`, `CapabilityError`).
